Scenario 3 and 4





In order to change the poster image of the saved project, the user has to get into their account and click on “my projects” in the menu. They have to choose which project they want to change the poster image of and click on “view project”. They then have to scroll down to “overview” and click on “add poster image”. The user can then select an image file. To submit a project without an application fee the user has to once again go through the filters after going to the “browse festivals” page. The user has to scroll down to the “Entry fees” section on the left where the user has to shorten the bar from the right to the left until the counter hits $0. The user can now select any of the filtered out festivals to submit their project to.

Scenario 5

This process only has a few steps. Once the festival in your cart is submitted you can withdraw the submission by going to “my projects”. The user then has to select the submitted project they want to delete and click on the “withdraw from festival” button on the right. The user will then see a pop-up saying that the withdrawing of the submission cannot be reversed. The user can then click on the “okay” button to finalize the withdrawal of the submission.
